### 중고폰 적정 가격 예측 서비스 챗봇 Almago 구현
#### 주요 내용
- 중고폰 적정 가격 예측 서비스 챗봇 ‘얼마고’ 구현  
- 45만건의 중고 거래 데이터를 분석하여 중고폰 모델별 적정 가격을 예측하고, 이를 카카오톡 API를 통해 구현  

#### 사용 기술
- Language : python, R
- AI/Big Data Analysis : ML
- Framework & Library : sklearn

#### 주요 역할
- 데이터 전처리 및 분석 알고리즘 구현
